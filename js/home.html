Important FAQ

What is the document object model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. Essentially, the DOM allows web pages to be dynamic by enabling JavaScript to interact with HTML and CSS.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript methods like `getElementById`, `getElementsByClassName`, `querySelector`, or `querySelectorAll`. These allow developers to target and manipulate elements based on their ID, class, tag, or CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element, rather than individual listeners to each child element. It allows events triggered by children to be captured by the parent. This is useful for managing dynamic content and improving performance, especially in large DOM trees.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like `setAttribute`, `getAttribute`, or directly through properties like `element.id` or `element.src`. For styles, you can change the `style` property of the element, such as `element.style.color = 'red';`, or by adding/removing CSS classes with `classList` methods.
